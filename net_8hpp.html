◆ tryConnect()

void net::tryConnect ( )

Try to connect to the network without blocking. This function will attempt to connect to the network and return immediately. If the connection is successful, connected() will eventually return true.

Note
This function can be safely called in a hard loop to keep trying to connect, but due to quirks in the WiFi library, it may occasionally block for a fraction of a second.
